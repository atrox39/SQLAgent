# SQL Agent

# Versions
`Python 3.10.6`

# Installation and usage of virtualenv

### Create virutal env
```bash
python -m venv venv
```

###
```bash
./venv/Scripts/activate.ps1
```

# Installation of dependencies
```bash
python -m pip install -r requirements.txt
```

# Execution
```bash
python server.py
```
